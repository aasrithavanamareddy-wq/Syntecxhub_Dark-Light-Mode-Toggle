# Syntecxhub_Dark-Light-Mode-Toggle
The Dark / Light Mode Toggle is a simple and interactive web development project that allows users to switch between dark and light themes on a website. The project is built using HTML, CSS, and JavaScript. It enhances the user experience by providing a customizable interface that reduces eye strain in low-light environments while maintaining readability in bright conditions.
The application includes a stylish toggle switch with icons (such as a sun and moon) to represent the two themes. When the user changes the theme, JavaScript dynamically updates the website's colors by adding or removing CSS classes. The selected theme is saved in the browser's localStorage, ensuring that the user's preference is remembered even after refreshing or reopening the page. Smooth CSS transitions are used to create an attractive and seamless switching effect.
